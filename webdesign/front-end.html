课程介绍

前端开发是创建 Web 页面或 APP 等前端界面呈现给用户的过程。它使用 HTML、CSS 和 JavaScript 以及衍生出来的各种技术、框架和解决方案，来实现互联网产品的用户界面交互。 简单来说，前端工作的目标是将产品的 UI 设计稿在用户终端上实现，因此用户看到的所有视觉和交互都是前端的工作领域
